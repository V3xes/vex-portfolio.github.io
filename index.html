📝 List Maker

List Maker is a retro-themed desktop app built using Python and Tkinter. It’s designed for organizing gift lists, shopping plans, or to-dos in a clean, nostalgic UI styled after old-school operating systems.

  • 💰 Automatic total cost calculation
  • 📁 Import/export support for saving and restoring lists
  • 🎮 A unique 8-bit styled UI inspired by classic PlayStation menus
  • 🖱 Button to open all item links at once in your browser
  • 📋 Supports multiple separate lists (like events or holidays)

This project merges functionality with nostalgic style. It's lightweight, polished, and great for managing gifts, shopping, or personal tasks with style.
